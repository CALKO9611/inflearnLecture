#### 📕 이런 걸 배워요!

1. 자바스크립트 기초 문법
2. CSS 기초 문법
3. 리액트 기초 탄탄하게 다지기
4. 미니 프로젝트를 통한 웹애플리케이션 개발

- JavaScript 기초 문법을 익힙니다.
- CSS의 개념과 자주 사용되는 속성을 배웁니다.
- 리액트에 대한 기초 개념을 쌓고 사용법을 익힙니다.
- 미니 프로젝트를 통해 실제 리액트 애플리케이션을 개발합니다.
- 입문자분들을 위해 HTML, CSS, JS의 기초 내용을 포함하고 있습니다.
